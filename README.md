Clone this repository :
```bash
git clone https://github.com/CSC105-2024/G20-GrabYourBook.git
cd G20-GrabYourBook
```
-----
How to run :
- Run Front-end only:
  ```bash
  npm run frontend
  ```
- Run Back-end only:
  ```bash
  npm run backend
  ```
- Run the project (front-end & back-end):
  ```bash
  npm run dev
  ```
